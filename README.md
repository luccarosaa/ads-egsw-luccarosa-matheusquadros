# ads-egsw-luccarosa-matheusquadros
Projeto
Esse repositório foi criado a partir do protótipo Nutria, um html (com estilização em CSS e lógica em JavaScript) de estrutura meramente visual criado para apresentar a proposta de ideação do projeto de extensão da equipe 5, uma das equipes da turma TADS2 de 2025.

Conforme descrito nos commits iniciais, os códigos aqui inseridos foram construídos a partir de elicitação de requisitos por parte da equipe Nutria, e da interação de aprimoramento (desenvolvimento incremental) com utilização de Inteligência Artificial. No caso, o mootor utilizado foi o Gemini, da google, em sua versão 2.5 PRO.

Após a inclusão dos arquivos separadamente, foi criada segunda branch para organização dos arquivos em pastas, para melhor navegação e manutenções futuras.
